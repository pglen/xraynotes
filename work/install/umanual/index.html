XrayNotes User Manual Main Page

 Welcome. XrayNotes is a medical application that is a fusion of an image annotation software, medical image viewer and image processor, a distributed medical event database, a query engine and a data server. XrayNotes has been built to modern user interface standards, where the focus off the application is to be as unobtrusive as possible to the work flow. While the XrayNotes application introduces new user interface elements, users who are familiar with Windows applications will feel familiar with the controls introduced by XrayNotes.

XrayNotes is divided into N subsystems:
  •  Image annotation
  •  DICOM subsystem
  • Patient management and event management
  The subsystems are connected, for instance a patient management allows one to select the patient and an event related to that patient, the image annotation will automatically be directed to the location where this particular patient's images are stored.
  Another example of the subsystem connectivity is that when one views a DICOM image the  picture can be annotated with the standard imaging tools that is giving immediate diagnostic/annotative powers to ordinary DICOM image.

  With XrayNotes's 3-D projection system, DICOM series can be viewed from various angles to give maximum diagnostic powers and value. With XrayNotes innovative prospective viewer one can achieve successful 3-D imaging even with a low-power computer.

  With XrayNotes's Crystal 3-D projection the DICOM images and series can be constructed to mimic a crystal composite of the slices. the layers of slices are superimposed on each other with decreasing degree of alpha blending, thus achieving a crystal like effect to view even the most miniscule details off the DICOM images.
