XrayNotes Resampling Algorythms
   To increase appearent computer speed,  XrayNotes works with the advanced computing capabilies of modern processors. When moving an object (or image) on screen, XrayNotes uses multi threading technology to display a 'preview' qualtity image while moving it, and a 'fine' quality image when the user stopped the interaction. This delivers better performance while not degrading the final view quality of the image.

  XrayNotes allows one to configure the kind of resampling algorythm to use in both 'preview' and 'fine' cases. Here is a list of algorythms one can chose from:
